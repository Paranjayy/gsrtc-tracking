## High Impact

1. Mobile overhaul.
2. Better tracking intelligence.
3. Trust and utility layer.
4. Custom empty states and error states.
5. Faster discovery flows.

## Mobile Overhaul Ideas

- Sticky search bar on scroll.
- Collapsible filter rail.
- Bottom dock navigation.
- Single-column stacked cards.
- Tap-friendly route cards.
- Better spacing on small screens.

## Tracking Intelligence Ideas

- Current location to next stop distance.
- ETA confidence or stability indicator.
- Better live signal quality meter.
- Cleaner event timeline.
- Map-first detail drawer.
- Route progress visualization.

## Trust and Utility Ideas

- Custom 404 page.
- Better loading skeletons.
- Inline errors instead of alerts.
- Save favorites and recent searches.
- Shareable tracking URLs.
- Search history and quick recall.

## Discovery Ideas

- Surprise me random route.
- Smart search suggestions.
- Quick route presets.
- Popular route clusters.
- Compare routes side by side.
- Source badges for GSRTC and Redbus.

## Content Expansion Ideas

- Route detail pages.
- Bus type pages.
- Carrier index pages.
- City pair pages.
- Favorites / saved routes page.
- Reading-room style recommendations page.

## Build Order Recommendation

1. Mobile overhaul.
2. Tracking intelligence.
3. Trust and utility.
4. Discovery features.
5. Content expansion.

## Notes

- Keep the existing virtual-socket tracking flow.
- Avoid adding random features before the shell is easier to use on mobile.
- Prefer features that improve clarity, speed, and confidence.
